## Anh tên là gì?  (你叫什麼名字?)

#### Example
    範例1.
    越南：Anh tên là gì?
    中文：你的名字是什麼？
    越南: Tôi tên là OO
    中文：我的名字是ＯＯ
    
附註：
tên: 名字
là: 是    
##### 基礎代名詞 (常見) 
    Anh: 你(我)
    Em：妳(我)
    Tôi: 我
    Mình: 我 (親切用法)
    Bạn:你(朋友)
